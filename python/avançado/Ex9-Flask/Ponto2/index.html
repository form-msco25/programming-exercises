Raspberry Pi 5

Principais características físicas

  • Processador Broadcom ARM Cortex-A76, 2.4 GHz
  • Até 8 GB de RAM DDR4
  • 2 portas USB 3.0 e 2 USB 2.0
  • 2x saída HDMI 4K
  • Ethernet Gigabit com PoE (Power over Ethernet)
  • WiFi 2.4 / 5 GHz e Bluetooth 5.0
  • Conector USB-C para alimentação

Fonte: Raspberry Pi Foundation
